[고객명]  귀하


 [고객명]님 귀하의 무궁한 발전을 기원합니다.

귀하와 [회사명] 사이에 체결된 금전소비대차계약(대부거래계약)에 있어 [회사명]이 귀하에 대하여 가지는 일체의 금전채권에 관한 권리를

(                                       )에 양도 하였는바, 이에 도의 채권양도 사실을 통지합니다.

주소

□ 귀하와 [회사명]사이에 체결된 대부거래약정서상의 제반 금융조건은 동일합니다.


- 채무자 (생년월일) : [고객명] ([주민번호]) / 보증인 (생년월일) : [보증인명] ([보증인주민번호])

- 대출잔액 : [대출잔액]  /  약정일 : [약정일]  / 대출이율 : [정상금리]  /  연체이율 : [연체금리]

- 대출계약일 : [대출일]  /  대출만기일 : [만기일]


□ 입금계좌는 양수인이 별도로 연락드리거나 가상계좌인 경우는 계좌는 동일하나 예금주가 변경될 수 있습니다


□ 채권양도에 따라 변경된 금전소비대차계약서를 교부할 수도 있으니 번거로우시더라도 동일한 계약조건을 기재하여 보내주시기바랍니다.


 위 사항과 관련하여 문의사항이 있으시면 채권양수인의 연락처로 문의 하시기 바랍니다. 감사합니다.
